## Getting Started

Add the right ENV keys to your `.env` / `.env.local`
S3_BUCKET_NAME
AWS_ACCESS_KEY_ID
AWS_SECRET_ACCESS_KEY

Run the development server:

```bash
pnpm dev
```

Then add [http://localhost:3000](http://localhost:3000) to your ChatGPT plugins (requires plugin access, sorry)
